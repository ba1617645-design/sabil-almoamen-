# سبيل المؤمن

موقع وتطبيق ويب إسلامي شامل يضم:
- القرآن الكريم بنظام الصفحات
- الأذكار اليومية
- الأدعية المأثورة
- مواقيت الصلاة
- الفتاوى والأسئلة الشائعة
- الختمة الفردية والجماعية

## ملفات الرفع إلى GitHub

ارفع هذه الملفات كما هي:
- `index.html`
- `index1.html`
- `manifest.webmanifest`
- `service-worker.js`
- `app-icon.svg`

## تشغيل GitHub Pages

1. ارفع الملفات إلى المستودع.
2. افتح `Settings`.
3. افتح `Pages`.
4. اختر:
   - `Deploy from a branch`
   - الفرع `main`
   - المجلد `/root`
5. احفظ الإعدادات.

بعد دقائق سيظهر رابط الموقع.

## ملاحظة

الملف الرئيسي للنشر هو `index.html`، وهو يفتح الموقع الفعلي الموجود في `index1.html`.
